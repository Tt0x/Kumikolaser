How To Lasercut Kumiko Designs

Used Software

  1. Webapp: https://www.kumikodesigner.com/
  2. Inkscape: https://inkscape.org/

Create your Kumiko design

  • With Kumiko Designer: https://www.kumikodesigner.com/
  • Export the Design as “black and white”
  • Save the design (.pdf) on your hard drive.

Prepare the File in Inscape

  • Open the software Inkscape
  • Open the saved .pdf File
  • Import the file with standard settings
Make sure to follow the steps one by one and don't mix it, otherwise it might not work as expected.
  1. Mark the whole Kumiko Design
  2. In the menu go to “Object” press “Ungroup”
  3. In the menu go to “Path” press “Object to Path”
  4. Go to “Path” and press “Exclusion”
  5. Unmark everything
  6. Press “n” on your keyboard
  • mark a line with your mouse
  • mark the rectangle
  • press delete
  • mark the small Line and delete it
Repeat until you deleted all the lines which should not be cut. Delete the intersections as well. If you delete too much use the undo function and try mark a different line or rectangle.
If you have two outer layers you can delete one.
  • Save the file with a different file name
  • Export the file as .svg

Import the .svg file into your laser cut software.

  • Mark everything and ungroup it.
  • Mark the parts you want to cut.

If you want the Kumiko in a bigger sheet of Wood mark the outside and set it to low power or engraving. So you don't cut the whole Kumiko out of the bigger sheet of wood.

Choose the right size, position and start the cutting process.
