How To Laser-Cut Kumiko Designs

Used Software

  1. Webapp: https://www.kumikodesigner.com/
  2. Inkscape: https://inkscape.org/

Create your Kumiko design

  • With Kumiko Designer: https://www.kumikodesigner.com/
  • Export the design as “black and white”
  • Save the design (.pdf) on your hard drive.

Prepare the File in Inkscape

  • Open the software Inkscape
  • Open the saved .pdf file
  • Import the file with standard settings
Make sure to follow the steps one by one and don't mix them up; otherwise, it might not work as expected.
  1. Select the whole Kumiko design
  2. In the menu, go to “Object” and press “Ungroup”
  3. In the menu, go to “Path” and press “Object to Path”
  4. Go to “Path” and press “Exclusion”
  5. Deselect everything
  6. Press “n” on your keyboard

  • Select a line with your mouse
  • Select the rectangle
  • Press delete
  • Select the small line and delete it
Repeat until you have deleted all the lines which should not be cut. Delete the intersections as well. If you delete too much, use the undo function and try to select a different line or rectangle.
If you have two outer layers, you can delete one.
  • Save the file with a different file name
  • Export the file as .svg

Finished File.

Import the .svg file into your laser cut software.

  • Select everything and ungroup it.
  • Select the parts you want to cut.

If you want the Kumiko in a bigger sheet of wood, select the outside and set it to low power or engraving. This way, you don't cut the whole Kumiko out of the bigger sheet of wood.

Choose the right size and position, then start the cutting process.
